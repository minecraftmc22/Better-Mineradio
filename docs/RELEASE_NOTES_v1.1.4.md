# Better-Mineradio v1.1.4 Release Notes

发布日期：2026-07-09

## 新增功能

### TTML 歌词与翻译支持
- 新增 Apple TTML 格式歌词解析器，支持逐词时间戳和翻译
- 歌词数据结构新增 `translation` 字段
- 支持网易云翻译歌词（tlyric）自动合并
- TTML 解析修复：空格处理（自动补全单词间缺失空格）

### 3D 歌词翻译显示
- 翻译歌词作为独立 3D mesh 渲染，带完整描边/发光效果
- 翻译位于主歌词下方，大小可调节
- 新增「翻译大小」滑块（设置 → 歌词 → 翻译）
- 翻译开关切换时实时重新渲染

### 桌面歌词翻译
- 桌面歌词新增翻译显示支持
- 新增「桌面歌词翻译」开关（设置 → 叠加效果）
- 翻译行样式：半透明白色，无描边

### 歌词源设置完善
- 歌词来源信息显示（平台标签 + 时间轴类型标签）
- 新增「自动搜索歌词」开关
- 新增「优先 TTML 歌词」开关

### 自定义歌词 TTML 支持
- 自定义歌词弹窗新增格式选择：LRC / TTML / 自动检测
- 自动检测模式：自动识别 TTML 和 LRC 格式

### 快捷键设置
- 新增「快捷键」设置区域（设置 → 快捷键）
- 支持配置：播放/暂停、上/下一首、歌词显示/隐藏、歌词放大/缩小、音量加/减、老板键
- 点击输入框后按下键盘组合键即可设置
- 默认快捷键：Space(播放)、MediaTrackPrevious/Next(切歌)、L(歌词)、Ctrl+/-/=(歌词大小)、ArrowUp/Down(音量)、Escape(老板键)

### 音量滚轮调节
- 鼠标悬停在音量按钮上时，使用滚轮调节音量
- 向上滚动 +5%，向下滚动 -5%

## Bug 修复

- 修复 TTML 歌词空格缺失问题
- 修复翻译 mesh opacity 未更新导致翻译不可见
- 修复残留的 updateLyricTranslationOverlay 调用导致 JS 报错
- 修复歌词描边层不跟随摄像头方向的问题

## 安装

下载 `Better-Mineradio-1.1.4-Setup.exe` 并运行即可。
